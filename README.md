# bio-inspired-artificial-intelligence
Collection of laboratories for the Bio-Inspired Artificial Intelligence course at the University of Messina

## Getting Started

Install the dependencies on a virtualenv on Linux/Unix as follows:
```bash
virtualenv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Install the dependencies on a virtualenv on Windows as follows:
```bash
python3 -m virtualenv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
```
